## Cómo instalar (How to install)


### Instalar por primera vez:

Ejecuta el siguiente comando en tu terminal (CMD/PowerShell):

```bash

pip  install  git+https://github.com/MatFunked/UCT-Python-Utils

```

  

### Actualizar a la última versión:

Si ya lo tenías instalado, actualízalo con:

```bash

pip  install  --upgrade  git+https://github.com/MatFunked/UCT-Python-Utils

```
